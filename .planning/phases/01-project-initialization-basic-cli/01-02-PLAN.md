       - Takes `dirPath` as an argument.
       - Resolves `dirPath` to an absolute path using `path.resolve()`.
       - Attempts to create the directory using `fs.promises.mkdir(absolutePath, { recursive: true })`.
       - Checks for write access using `fs.promises.access(absolutePath, constants.W_OK)`.

         - If `error.code === 'EACCES'`, throw a new `Error` with a message like "Permission denied: Cannot write to '{absolutePath}'. Please check folder permissions."

         - For any other error, throw a generic error message "Failed to access or create output directory '{absolutePath}': {error.message}".
    4. Export `validateAndPrepareOutputDir` from `src/utils/fs-utils.js`.
