Refer to "Pattern 1: URL Construction for Directory Listing" in the research.
    2. Implement the `getKlineZipFileUrls` asynchronous function. This function should:
      - Accept `symbol` (string) and `interval` (string) as parameters.
      - Use `getKlineDiscoveryUrl` to construct the `discoveryUrl`.
      - Fetch the HTML content from the `discoveryUrl` using `node-fetch`. Implement error handling for non-OK HTTP responses.
      - Parse the fetched HTML using `cheerio.load()`.
      - Iterate over all `<a>` (anchor) tags in the parsed HTML.
      - For each `<a>` tag, extract the `href` attribute.
      - Filter these `href` values: include only those ending with `.zip` and explicitly exclude those ending with `.zip.CHECKSUM`.
      - Construct full, absolute URLs for the filtered `.zip` files by combining the `href` with the `discoveryUrl` using `new URL(relativePath, baseUrl)`.
      - Return an array of these full `.zip` file URLs.
    3. Ensure proper `import` statements for `fetch` and `cheerio` are at the top of the file.
