  <name>Task 1: Implement DownloadService</name>
  <files>src/services/download.service.ts</files>
  <action>
    Create `DownloadService` class in `src/services/download.service.ts`.
    Implement `downloadFile(url: string, destPath: string): Promise<void>`:
    1. Validate inputs.
    2. Use `node-fetch` to GET the URL with a streaming response.
    3. Ensure the destination directory exists using `fs.mkdir(path.dirname(destPath), { recursive: true })`.
    4. Use `pipeline` from `node:stream/promises` to stream `response.body` to a `fs.createWriteStream(destPath)`.
    5. Wrap in `try/catch` to handle errors:
       - On failure, close the write stream and use `fs.unlink(destPath)` to remove partial downloads.
       - Re-throw the error with context.
    
    Implementation details from 03-RESEARCH.md: Use `node-fetch`, `pipeline`, and built-in `fs` modules.
  </action>
  <verify>
    Create a temporary script to test the service manually or run a quick TS check:
    `npx ts-node -e 'import { DownloadService } from "./src/services/download.service"; console.log(DownloadService)'`
  </verify>
  <done>
    `DownloadService` is implemented with streaming, recursive directory creation, and error cleanup logic.
  </done>
